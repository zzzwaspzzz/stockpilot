Stockpilot

Aplicación de escritorio para la gestión de almacenes.
Desarrollada en Java con Hibernate y MariaDB.

⬇ Descargar Stockpilot

Versión 1.0 · Windows · Requiere Java 8+

Requisitos previos

  • Java Runtime Environment 8 o superior — descargar en java.com
  • MariaDB instalado y en ejecución — descargar en mariadb.org

Instalación

  1. Descarga el instalador y ejecútalo. Sigue el asistente de instalación.
  2. Importa el archivo stockpilot.sql incluido en tu servidor MariaDB.
  3. Abre el archivo hibernate.cfg.xml y configura tu usuario y contraseña de MariaDB.
  4. Ejecuta Stockpilot desde el acceso directo del escritorio.
